📘 HTML Notes: Tables and Forms

📊 HTML Tables

HTML tables are used to display tabular data in rows and columns.

Table Syntax Example

<table>
  <tr>
    <th>Name</th>
    <th>Email</th>
    <th>DOB</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>alice@example.com</td>
    <td>01/01/2000</td>
  </tr>
</table>

You can create more rows using <tr> and define headers using <th>, and data using <td>.

📝 HTML Forms

Forms are used to collect data from users. They consist of various input types like text, email, password, checkbox, radio, dropdowns, etc.

Form Preview

📚 Input Types You Should Know

  • text – Basic single-line input
  • email – Validates email format
  • password – Hides input characters
  • checkbox – Allows multiple selections
  • radio – Allows one selection in a group
  • date, month, time – Date/time pickers
  • file – Uploads files
  • color – Color picker

🔍 How to Explore HTML Elements

To discover new elements and attributes:

  1. Search MDN documentation for the tag (e.g., MDN input email).
  2. Use browser inspect tools to understand structure.
  3. Experiment and test HTML in code playgrounds or local files.

✅ Summary

  • Tables are used to display structured data in rows/columns.
  • Forms collect user input using various input types.
  • Labels improve accessibility and form clarity.
  • Use name, id, type, and placeholder attributes effectively.
