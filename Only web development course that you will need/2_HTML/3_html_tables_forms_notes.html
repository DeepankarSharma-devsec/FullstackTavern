HTML tables are used to display tabular data in rows and columns.

Table Syntax Example

<table>
  <tr>
    <th>Name</th>
    <th>Email</th>
    <th>DOB</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>alice@example.com</td>
    <td>01/01/2000</td>
  </tr>
</table>

You can create more rows using <tr> and define headers using <th>, and data using <td>.
