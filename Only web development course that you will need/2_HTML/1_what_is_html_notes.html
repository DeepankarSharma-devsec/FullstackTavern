HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages. HTML structures content on the web and is readable by browsers, screen readers, and many other systems.

HTML was initially created to help scientists share research papers online. It allowed the inclusion of elements such as headings, paragraphs, and tables for clarity and accessibility.

HTML uses tags to define elements. Most tags come in pairs:

Here, <h1> is the opening tag, and </h1> is the closing tag.
