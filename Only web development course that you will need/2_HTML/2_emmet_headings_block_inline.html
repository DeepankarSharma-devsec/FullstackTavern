HTML Notes: Emmet, Headings & Block vs Inline

📌 Emmet Abbreviations

Emmet is a shorthand syntax that allows faster writing of HTML/CSS.

  • html:5 → Full HTML5 boilerplate
  • h1>lorem5 → Creates an H1 tag with 5 lorem ipsum words
  • ul>li*3 → Creates a ul with 3 li items
  • img:src → Creates an img tag with src

🧱 Headings and Paragraphs

HTML has six levels of headings:

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
...
<h6>Smallest Heading</h6>

Use <p> for paragraphs. Use Emmet: p>lorem100.

🖼️ Images

Basic syntax:

<img src="images/sample.png" alt="Description" width="300" />

🔗 Anchor Tags

<a href="https://example.com">Visit Site</a>
Special links:
  • Email: mailto:someone@example.com
  • Phone: tel:+1234567890

🧾 Lists

Two common types of lists:

  • Unordered List (ul)
  • Ordered List (ol)

📦 Block vs Inline Elements

Block Elements

Heading (h1 - h6)
Paragraph (p)
List (ul, ol, li)
Div

Inline Elements

a img strong em span

🧪 Block vs Inline Behavior

Block elements take full width, start on a new line. Inline elements only take required space.

Example:

This is a block:

Hello world!

This is inline: word1 word2

🧠 Important Definitions

  • Element: Full tag + content (e.g., <p>Text</p>)
  • Tag: Only the tag (e.g., <p>)
  • Attributes: Extra info inside tag (e.g., src=, href=)
  • Tooltips: Use title="..." to add on hover text

🔍 Developer Tools Tip

Use Chrome DevTools (Inspect Element) to check if an element is block or inline.

  • <div>, <p>, <h1> → show full-width block
  • <img>, <a>, <strong> → show only content width

✅ Quick Summary

  • Use Emmet to speed up HTML writing
  • Understand semantic differences between headings and paragraphs
  • Know which tags are block vs inline
  • Use attributes like src, href, title appropriately
