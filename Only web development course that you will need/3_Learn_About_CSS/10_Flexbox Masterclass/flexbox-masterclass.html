Flexbox Masterclass

This masterclass is a deep dive into the powerful CSS layout module called Flexbox. Below are the core concepts, properties, and usage examples explained with annotated diagrams.

üìå Key Concepts

  • Main Axis: Default is horizontal (left to right) in web.
  • Cross Axis: Vertical axis (top to bottom).
  • Properties are either container-level or item-level.

üìÅ Flex Container Properties

Diagram: Main Axis (Horizontal) and Cross Axis (Vertical)

1. display: flex

Applies Flexbox behavior to the container.

2. flex-direction

  • row (default)
  • column
  • row-reverse
  • column-reverse

3. justify-content (Main Axis)

  • flex-start, center, flex-end
  • space-between, space-around, space-evenly

4. align-items (Cross Axis)

  • flex-start, center, flex-end, stretch
  • baseline - aligns text baselines

5. flex-wrap

  • nowrap (default)
  • wrap - wraps items to next line if space runs out
  • wrap-reverse

Diagram: Using flex-wrap to prevent content squishing

6. align-content

Used when content wraps; controls vertical alignment of rows (cross axis).

üéØ Flex Item Properties

7. order

Rearranges the visual order of items without changing HTML.

8. flex-grow

Controls how much of remaining space an item should take.

9. align-self

Overrides align-items on individual items.
