Twitter Style Navbar in CSS

Overview

This project demonstrates a vertical navigation bar inspired by Twitter's sidebar layout using HTML and CSS. Key concepts covered include Flexbox layout, block-level styling, hover effects, and transition properties.

HTML Structure

<div class="nav-container">
  <h2>Twitter Style Navbar</h2>
  <nav class="nav-two">
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">Explore</a></li>
      <li><a href="#">Notifications</a></li>
      <li><a href="#">Messages</a></li>
      <li><button class="login-btn">Login</button></li>
    </ul>
  </nav>
</div>

CSS Key Concepts

Sidebar Layout (Vertical Navbar)
  • Fixed width (200px)
  • Unordered list for navigation items
  • Full width clickable items using display: block and width: 100%
Hover Effects & Transitions
  • :hover used to change background color
  • transition enables smooth effect on hover
  • Buttons styled separately with hover states

CSS Snippet for Hover

.nav-two a:hover {
  background-color: #2c3136;
  transition: background-color 0.3s ease;
}

.login-btn:hover {
  background-color: #2e8b57;
  border: 1px solid #ffffff;
}

Final Thoughts

This project focused on creating a Twitter-style vertical navbar with CSS fundamentals. While professional projects may use component libraries like Tailwind CSS, building from scratch helps understand CSS deeply and fix issues independently.
