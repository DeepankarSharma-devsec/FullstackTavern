Box Model, Inline and Block Elements

1. Introduction to Box Model

The box model is foundational in CSS and governs how much space an element consumes on a webpage. It includes content, padding, border, and margin.

Box Model Layers:

  • Content
  • Padding
  • Border
  • Margin

2. Types of Box Models

There are two primary types of box models:

  • Content-box: Width and height only include the content. Padding and border are added outside.
  • Border-box: Width and height include content, padding, and border.

Illustration of Width Calculation:

Content-box: Total width = content + padding + border

Border-box: Total width = defined box size (content + padding + border included)

3. Inline vs Block Elements

Inline Elements: Do not start on a new line and only take as much width as necessary.

Block Elements: Always start on a new line and take up the full width available.

4. Summary

  • Box model includes: content, padding, border, margin
  • Content-box adds padding and border outside width/height
  • Border-box includes padding and border inside width/height
  • Inline elements flow within the text line
  • Block elements take full width and break the line
