CSS Selectors Masterclass

What are CSS Selectors?

CSS selectors are patterns used to select the elements you want to style.

Types of Selectors

  • Universal Selector: * selects all elements.
  • Type Selector: Targets elements by tag name (e.g., p, h1).
  • Class Selector: Uses a dot prefix to target elements (e.g., .highlight).
  • ID Selector: Uses a hash prefix (e.g., #header).
  • Attribute Selector: Targets elements with specific attributes (e.g., input[type="text"]).
  • Descendant Selector: Targets all descendants (e.g., article p).
  • Child Selector: Targets direct children only (e.g., div > p).
  • Adjacent Sibling Selector: Targets element immediately after another (e.g., h1 + p).
  • General Sibling Selector: Targets all siblings after a specific element (e.g., h2 ~ p).
  • Pseudo-Class Selector: Based on user interaction (e.g., a:hover).
  • Pseudo-Element Selector: Targets part of an element (e.g., p::first-letter).
  • Grouping Selector: Apply styles to multiple selectors (e.g., h1, h2).

Important Notes

  • Always read selectors from right to left.
  • Use classes and IDs for precision and maintainability.
  • Universal selector is often used to reset margin and padding.
  • Tailwind CSS follows atomic CSS principles used by Facebook.
