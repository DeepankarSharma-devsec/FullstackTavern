🎯 Assignment: Build Beautiful Cards (HTML + CSS)

Explore creativity, flexbox, and design principles through this assignment.

📝 Objective

Build 4 unique cards using HTML and CSS with creative freedom. Each card should follow this structure:

  • Background: Image / Gradient / Solid Color
  • Card Contents:
    • Image (Top or Centered)
    • Title
    • Subtitle
    • Two Buttons (e.g., Buy Now and Add to Wishlist — icons allowed)

🎨 Inspiration Sources

Use the following for ideas and components:

  • Font Awesome – for free web icons
  • CodePen – browse card examples using HTML & CSS

CodePen is a playground to explore card layouts from real developers.

💡 Tips

  • Use display: flex or display: grid to align elements
  • Feel free to modify designs, use icons, or add animations
  • Don't copy-paste — understand and adapt the code

📷 Submission

Once completed, take screenshots or a selfie with your work and share:

  • 📍 Twitter: Tag your instructor
  • 📍 Discord: Share with fellow learners
  • 📍 LinkedIn: Show off your card designs!
