3. Grid Template Columns and Rows

Using grid-template-columns and grid-template-rows to define structure.

grid-template-columns: repeat(3, 1fr);
grid-template-rows: repeat(3, 100px);

4. Grid with Different Column Sizes

Fractions can be allocated differently: e.g., 1fr 2fr 1fr

5. Grid with Gaps

Add spacing using gap: 20px; or more depending on design.

6. Grid Item Placement

Customize an item's position using grid-column and grid-row properties.

7. Grid Template Areas

Use names for layout regions for better readability.

grid-template-areas:
  "header header header"
  "sidebar main main"
  "footer footer footer";

8. Autofit & Minmax

Let the grid decide the best layout with repeat(auto-fit, minmax(100px, 1fr)).

9. Alignment & Justification

Grid supports justify-content and align-items for layout alignment.

10. Nested Grid

A grid container can contain another grid container for modular layouts.

11. Grid Auto Rows

Use grid-auto-rows: minmax(100px, auto) to define the behavior of implicit rows.

Summary

  • Most Used: grid-template, repeat, fraction units, gaps
  • Rarely Used: grid-auto-rows
  • Must Know: grid-template-areas, autofit/minmax, nested grids
