Lecture Notes

Title: Build a Custom Coming Soon Template Project

Date: June 26, 2025

Goals:

  • Build a "Coming Soon" landing page using only HTML and CSS.
  • Understand how real-world freelance projects are handled.
  • Focus on simplicity, structure, and reusable design principles.

Project Setup:

  • Created a file named coming-soon-landing.html.
  • Created a separate style.css file and linked it in the HTML head.
  • Set up basic HTML boilerplate.
  • Used semantic div structure: container > content.

HTML Structure:

  • Container with a nested content block.
  • Content includes:
    • Company logo (image)
    • <h1>Coming Soon</h1>
    • <p>We are working hard...Stay tuned!</p>
    • Email collection form with:
      • Email input field (required)
      • Submit button labeled “Notify Me”

CSS Highlights:

  • CSS Reset using margin: 0; and padding: 0;
  • Font family set to Arial/Helvetica and later updated via Google Fonts.
  • Used linear gradients for background (e.g. 135° with 2 shades of purple).
  • Applied Flexbox to center content both vertically and horizontally.
  • Max-width and padding used to constrain and space content appropriately.
  • Used rgba() for transparent white background on content block.
  • Added border-radius and box-shadow for clean look.
  • Email input:
    • Styled with padding, border-radius, and soft gray border.
  • Button:
    • Purple background, white text, no border, border-radius: 5px.
    • Hover effect: background color change with smooth transition.
    • cursor: pointer; for better UX.

Advanced Additions:

  • Google Fonts integration (example: Noto Sans).
  • Discussed SVG wave generators for fancy footer design.
  • Explained position: absolute; usage with SVG elements.
  • Recommended exploring box-shadow properties via MDN Docs.

Developer Tips:

  • Test styling via body { background: red; } before full implementation.
  • Use meaningful classes (e.g. .logo) for styling images.
  • Prefer dashes in IDs and classes for readability (notify-form).
  • REM vs PX: REM is relative to root font-size, PX is absolute.

Assignments Mentioned:

  • Customize the focus color on the email input field.
  • Position SVG wave graphic properly at the bottom.

Conclusion:

This hands-on lesson demonstrated how real-world projects like freelance landing pages are structured, coded, and styled from scratch using clean, semantic HTML and scalable CSS techniques. Simplicity and customization were key takeaways.
