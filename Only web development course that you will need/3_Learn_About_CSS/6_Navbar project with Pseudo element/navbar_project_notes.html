Navbar Project with Pseudo Element

Overview

This project demonstrates how to build a responsive navigation bar using HTML, CSS Flexbox, and CSS pseudo-elements like ::after.

HTML Structure

<div class="nav-container">
  <h2>Navigation Bar</h2>
  <nav class="nav-one">
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Products</a></li>
      <li><button class="login-button">Login</button></li>
    </ul>
  </nav>
</div>

Flexbox Layout

The navigation menu uses display: flex to arrange the li elements horizontally.

Flex Layout:
  • Parent: display: flex
  • Child: flex items (nav links)
  • Justify-content: space-between or space-around

Styling the Nav and Buttons

Basic styling includes:

  • Removing list bullets: list-style-type: none
  • Spacing: padding and margins
  • Color and background customization
  • Button style: border-radius, background, hover

Pseudo-element Usage (::after)

The ::after pseudo-element is used for hover underline animations.

.nav-one a:hover::after {
  content: "";
  position: absolute;
  left: 0;
  bottom: 0;
  height: 2px;
  width: 100%;
  background-color: #007BFF;
  transform: scaleX(1);
  transition: transform 0.3s ease-in-out;
}

.nav-one a::after {
  content: "";
  position: absolute;
  left: 0;
  bottom: 0;
  height: 2px;
  width: 100%;
  background-color: #007BFF;
  transform: scaleX(0);
  transition: transform 0.3s ease-in-out;
}
Diagram:

Pseudo-element is placed at the bottom of the <a> and animated using transform: scaleX.

Debugging Notes

If the pseudo-element is not appearing correctly:

  • Ensure parent element has position: relative
  • Pseudo-element must have content defined
  • Ensure pseudo-element has size (width/height)

Final Thoughts

Using ::after creatively allows for beautiful animations and interactivity in navigation bars. This project demonstrates both CSS basics and advanced techniques like transitions, transform, and pseudo-elements.
