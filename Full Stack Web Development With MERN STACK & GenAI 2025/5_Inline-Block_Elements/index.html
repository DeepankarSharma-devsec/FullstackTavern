This is a paragraph. It is a block-level element, which means it takes up the full width available and starts on a new line.

This is a div element. It is also a block-level element and takes up the full width available. It starts on a new line.
This is an inline element. It does not start on a new line and only takes up as much width as necessary. Another inline element. This is a link. It is also an inline element and does not start on a new line. This is an underlined text. It is an inline element as well.
